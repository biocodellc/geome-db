Genomic Observatory MetaDatabase (GeOMe) Field Information Management System (FIMS)

GeOMe has developed a web-based database to aid collaboration and the cross-dissemination of published data. Although Genbank is an important repository of sequence data, it does not require the uploading of important metadata associated with individual genotypes such as precise sampling locations and there is no central repository nor set standards for such population genetic data and associated metadata. The GeOMe database is designed to fill this gap.

Data is subject to our data usage policy found here

Members can upload two types of data (under "tools" tab)

  1. Published Sanger sequence data and the associated metadata (fasta format)
  2. Metadata associated with next-generation sequence data (fastq format)
    • In the process of uploading the metadata for your next-generation sequence data we create two files:
      1. the SRA metadata file
      2. the BioSample datasheet
       

This will make uploading your data to Genbank's Short Read Archive (SRA) database faster and easier.

* Note we are not currently accepting microsatellite datasets

What you need to do before you upload data

  • Generate a FIMS template (under “tools” tab) for your metadata
    • This option allows the user to create an excel file to store the metadata associated with each of their samples.
